Detailed Description

Stores a location URI.

Attributes should not be modified after instantiation.

See the following for specifications: http://en.wikipedia.org/wiki/URI_scheme#Generic_syntax http://tools.ietf.org/html/rfc3986

This class is nearly identical to urlparse.ParseResult, with a few differences.

  1. This class does not use "named tuples" which are nearly identical to a dict. It is believed that named tuples were created for backwards compatibility with that class, which is being dropped here.
  2. ParseResult is read only, whereas this class could more easily be expanded in the future to allow writing to instance attributes.
  3. This class uses "authority" instead of "netloc" because "authority" is the term used in RFC 3986.
